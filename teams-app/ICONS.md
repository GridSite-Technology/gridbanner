# Icon Requirements

The Teams app requires two icon files:

1. **icon-color.png**: 192x192 pixels, color icon
2. **icon-outline.png**: 32x32 pixels, outline icon

## Creating Icons

You can create these icons using any image editor:

1. Create a 192x192 pixel image with the GridBanner logo/branding for `icon-color.png`
2. Create a 32x32 pixel simplified outline version for `icon-outline.png`

## Quick Icon Creation

If you don't have icons ready, you can:
- Use a placeholder image generator
- Create simple colored squares with text
- Use the GridBanner branding colors (#00ADEE)

The icons should represent the GridBanner/alert management functionality.
