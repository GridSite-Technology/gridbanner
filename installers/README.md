# Installers (Windows x64)

This folder contains WiX-based MSI installers for:

- `GridBanner` (the banner agent)
- `BannerManager` (the alert editor / trigger tool)

## Silent install examples

```powershell
msiexec /i GridBanner.msi /qn /norestart
msiexec /i BannerManager.msi /qn /norestart
```
